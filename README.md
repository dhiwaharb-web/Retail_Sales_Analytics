# Retail Sales Analytics

## Project Overview

This project analyzes retail transaction data using Python and MySQL to identify sales trends, top-performing products, customer behavior, and country-wise revenue performance.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* MySQL
* Jupyter Notebook

## Analysis Performed

### Python

* Data Cleaning
* Missing Value Analysis
* Revenue Calculation
* Product Analysis
* Customer Analysis
* Country Analysis
* Monthly Revenue Trend

### SQL

* Total Revenue
* Total Orders
* Total Customers
* Average Order Value
* Top Customers by Revenue
* Top Products by Revenue
* Revenue by Country
* Customer Ranking
* Customer Segmentation
* Monthly Revenue Trend
* Month-over-Month Growth Analysis

## Key Findings

* United Kingdom generated the highest revenue.
* Revenue peaked during November 2011.
* A small group of customers contributed a large portion of sales.
* Several products consistently generated the highest revenue.

## Files

* `Retail_Sales_Customer_Analytics.ipynb` – Python analysis
* `Retail_Sales_SQL_Analysis.sql` – SQL queries
